# Text extracted from Shakespeare Poem

```
FairySong
Overhill,overdale,
Thoroughbush,thoroughbrier,
Overpark,overpale,
Thoroughflood,thoroughfire!
Idowandereverywhere,
Swifterthanthemoon'ssphere;
AndIservetheFairyQueen,
Todewherorbsuponthegreen;
Thecowslipstallherpensionersbe;
Intheirgoldcoatsspotsyousee;
Thoseberubies,fairyfavours;
Inthosefreckles'livetheirsavours;
Imustgoseeksomedewdropshere,
Andhangapearlineverycowslip'sear.
From"AMidsummerNight'sDream,"ActII.Sc.1.
ByWILLIAMSHAKESPEARE
```
